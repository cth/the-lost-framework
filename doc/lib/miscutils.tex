Various useful utility predicates that doesn't fit anywhere else.\vspace{0.7cm}

\begin{description}
    \predicate[private]{terms_has_rule_with_head}{3}{+Terms, +Functor, +Arity}
True if the list \arg{Terms} has a rule with a given \arg{Functor} and \arg{Arity}

    \predicate[private]{atom_integer}{2}{?Atom, ?Integer}
Converts between atom representing an integer number to an
integer usuable in arithmetic operationes and vice versa
\end{description}
